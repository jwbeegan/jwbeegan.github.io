Below are the jam games I helped create, each with a different team. I enjoy game jams because they give me the oppurtunity to test my skills in a more collaborative environment. Creating understandable and flexible interfaces for designers, implemeting SFX/music from musicians and assisting with animation control for artists are all skills I have used to make these games.

Sundai Showdown (GMTK Game Jam 2023)

Play Sundai Showdown Here

Sundai Showdown is a combo game about quick reactions. In the first phase, the player must input combos shown on the left as fast as they can before the timer runs out. Each round the time they have to input the required combos decreases. In the second phase, the player must react to those same combos and match the inputs. If they do not, they are hit. If not enough combos are inputted or if the player takes too many hits, the game is over.

Contributions:

  • Combo input phase
  • Combo playback phase
  • Combo creation interface for designers
  • Player input
  • Camera management
  • Game state manager
  • Animation control

Go Fish! (GMTK Game Jam 2022)

Play Go Fish! Here

Go Fish! is a relaxing take on the card game Go Fish. The first phase plays out like a regular game of Go Fish, but instead of grabbing a card from the deck, the player must play a mini game. In this mini game the player controls a descending hook to reach the bottom and grab higher value fish.

Contributions:

  • Game state management
  • Hook gameplay
  • Go Fish card game implementation
  • Camera management
