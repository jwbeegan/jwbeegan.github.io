MechAvenger is a VR shooter platformer, which I released January 10th, 2024. This is my first steam release and my first published game, outside of game jams.

I implemented the following features for MechAvenger:

  • Multiple VR controlled weapons with:
    • Projectile types such as standard bullets, shotgun shells, grenades, and rockets
    • Reloading methods such as clips and internal magazines
    • Iron sights and zoomed in scopes
    • Semi-Automatic, fully automatic and burst fire rates
    • Varying damage, ammo types, and effects
  • A variety of enemies:
    • Slow melee enemies
    • Fast melee enemies
    • Ranged enemies
    • Turrets
    • Shielded enemy variants
  • platforming and player movement systems:
    • Sprinting
    • Jumping
    • Climbing
    • Grapple hook
  • VR physics object grabbing system for two handed and one handed objects
  • 7 different levels which mix platforming and combat
  • Melee attacks with both fists and weapons
  • Quick return holster system which returns weapons to holsters automatically
  • Enemy animation control
  • Quick reload system in which clips are automatically loaded into a weapon from a belt
  • Health, pause, and save systems
  • Audio for weapon sounds, enemy sounds and soundtrack

programming methods and engine functions used:

  • Created with Unity
  • Use of custom classes to create standard interfaces
  • Finite state machine for enemy AI, animation and object grabbing system
  • Unity physics to control objects and player
  • Lighting and collider optimization
  • Particle systems
  • OpenXR toolkit for VR implementation and control
  • Input system for cross platform compatability
  • Raycasts to implement weapons and enemy AI
  • Pro Builder to quickly create and revise levels
